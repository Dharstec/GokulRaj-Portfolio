Introduction

This project focuses on the development of a simulated barista cup holder robot model using advanced robotics modeling techniques. By leveraging URDF and XACRO macros, we aim to create a modular and maintainable robot design suitable for dynamic simulation in Gazebo, powered by ROS2.

Objectives
  • To model the robots using Unified Robot Description Format (URDF).
  • Enhance the design with XACRO macros for modular and maintainable code.
Tools and Techniques:
  • Programming Languages: XML for URDF/XACRO
  • Frameworks and Libraries: ROS2, URDF
  • Simulation Environment: Gazebo
  • Version Control: Git

Base Robot Model Creation

Initial Setup: The project commenced with cloning the essential repositories and configuring the Git environment tailored for the project requirements.

Robot Modeling: A detailed robot model for Rick and Morty was constructed using URDF, including elements like the mobile robot base, top plate, and sensory equipment such as a laser scanner and wheel encoders. The parts of the robot were designed in Autodesk Fusion based on precise dimensions, and textures were applied using Blender.
The final model was validated in RVIZ to ensure the correct position of link frames and laser sensor functionality.(ViewMore...)

Refinement with XACRO Macros

XACRO Implementation: The URDF model was transitioned to XACRO to utilize macros for a more modular and maintainable structure, facilitating easier updates and alterations to the robot model.

Simulation Environment Setup: Gazebo was used for the simulation environment, ensuring accurate robot movements controlled via the differential drive plugin.(ViewMore...)

Results

The culmination of this project is a fully functional simulation of the barista robot, showcasing the flexibility and capabilities of ROS2 alongside URDF and XACRO in robotic modelling.

Key Insights
  • The project highlights the significance of modular design in robotics, showcasing the benefits of using XACRO over plain URDF for complex models.
  • Adding a negligible inertia to reference links is necessary to avoid their fusion with other links in Gazebo.

This project not only demonstrates the advanced capabilities of ROS2, URDF, and XACRO but also provides a foundation for future developments in dynamic and interactive robotic simulations.

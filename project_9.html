Introduction

This project involves developing a robotic arm capable of performing pick and place tasks. Initially, we configured a UR3e robotic arm using MoveIt2 to execute a basic pick and place sequence. In the next phase, we integrated a perception system to enhance the robot's ability to detect and locate objects, making the task more robust and adaptable.

Objectives
  • Set up and configure the MoveIt2 package for the UR3e robotic arm.
  • Develop a C++ program to generate a pick and place sequence using MoveIt2's Move Group Interface API.
  • Extend the pick and place task by incorporating a perception node to detect and locate objects on the table.
  • Test and validate the system both in simulation and on a real UR3e robotic arm.
Tools and Technology
  • MoveIt2 for motion planning and control.
  • ROS2 for robot operating system middleware.
  • C++ for developing the pick and place program.
  • RViz2 for visualizing the robot's state and perception data.
  • Gazebo for robot simulation.
  • UR3e robotic arm hardware for real-world testing.
  • Perception systems and sensors for object detection.

Process and Development

Without Perception

Initially, we set up the environment by creating and configuring a MoveIt2 package for the UR3e arm using the provided URDF file [Image]. We then verified that the MoveIt2 configuration could control the robot in the simulation[Image]. Following this, we developed a C++ program (pick_and_place.cpp) that utilized the Move Group Interface API to execute a series of motions for the pick and place task. This program was tested in a simulated environment using predefined object positions. Subsequently, we created a new MoveIt2 package for the real UR3e arm and tested the program in a real robot lab session to ensure it performed the pick and place task accurately.[Video]

With Perception

To incorporate perception, we developed a perception node capable of detecting the object's position using point cloud data from a camera. We visualized the detected object and support clouds in RViz2 to ensure accurate detection [Image]. The pick and place program was then modified to integrate the perception node, allowing the robot to dynamically determine the object's position. This integrated system was tested in simulation to ensure the robot could pick and place objects based on real-time perception data. Finally, the enhanced system was tested in the real robot lab, verifying that the perception-enhanced pick and place task worked correctly with the real UR3e arm.[Video]

Results

TIn simulation, we successfully configured and controlled the UR3e robotic arm using MoveIt2, executing the basic pick and place task with predefined positions. The task was further enhanced with a perception system, allowing dynamic object detection and pick-up. In the real-world testing phase, we verified the pick and place sequence with the real UR3e arm and demonstrated the integration of perception, enabling the robot to adjust to different object positions in a real environment.
This project highlights the successful implementation of a robust pick and place task using the UR3e robotic arm, enhanced by integrating a perception system for dynamic object detection. The developed system shows promise for various applications in industrial automation and robotics.

Key Insights
  • Integrating perception significantly improves the flexibility and robustness of robotic pick and place tasks by allowing dynamic adaptation to varying object positions.
  • The combination of MoveIt2 and ROS2 provides a powerful framework for developing and simulating complex robotic tasks before deploying them to real hardware.
  • This project demonstrates the potential for advanced automation in industrial settings, highlighting the importance of perception systems in enhancing robotic capabilities.

This project not only demonstrates the advanced capabilities of ROS2, URDF, and XACRO but also provides a foundation for future developments in dynamic and interactive robotic simulations.

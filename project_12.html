Project Description:

This project focuses on designing and optimizing PID controllers for the Rosbot XL robot to achieve precise navigation within the CyberLab environment. We will develop a Distance PID Controller for accurate movement and stopping at specific distances, along with an Orientation PID Controller to ensure precise alignment with waypoints. The main goal of this project is to create a robust system capable of autonomously navigating complex mazes. Through rigorous simulation and real-world testing, this project aims to showcase the effectiveness of PID control in enhancing the navigation capabilities of mobile robots, paving the way for advanced autonomous systems in robotics.

Objectives
  • Acquire the coordinates of each waypoint within the maze.
  • Develop a Distance PID Controller to ensure accurate movement and stopping at predefined distances.
  • Create an Orientation PID Controller for precise alignment with waypoints.
  • Implement these controllers using ROS2 and C++ in conjunction with the Gazebo simulator.
  • Test and refine the system through simulation and real-world scenarios to demonstrate its efficacy in navigating complex mazes autonomously.
Tools and Techniques:
  • Robot Operating System (ROS2): Industry-standard middleware for robotics.
  • C++ programming language: Efficient for implementing control algorithms.
  • Gazebo simulator: Provides a realistic 3D simulation environment.
  • CyberLab environment with Rosbot XL: Real hardware setup for testing.

Results

The project outcomes highlight enhanced navigation precision and efficiency of the Rosbot XL, as it solves the maze environment, demonstrating the effectiveness of PID controllers in autonomous navigation tasks.

Key Insights
  • PID Controller Versatility: Demonstrates the broad applicability of PID controllers beyond robotic arms, showcasing their effectiveness in mobile robot navigation.
  • Importance of Fine-Tuning: Emphasizes the critical role of PID parameter optimization in achieving both speed and precision in navigation tasks.
  • Iterative Development Process: Highlights the iterative approach of testing in both simulation and real-world environments to ensure robust and reliable control algorithms.

This project not only demonstrates the advanced capabilities of ROS2, URDF, and XACRO but also provides a foundation for future developments in dynamic and interactive robotic simulations.

Mobile Robot Path Planning using Nav2: Implementation of Dijkstra's Algorithm

Implementing Autonomous Navigation in Industrial Robots Using Dijkstra's Algorithm

GIT LINK 1

Introduction

This project aims to enhance the autonomous navigation capabilities of a state-of-the-art industrial robot within a factory setting. By implementing Dijkstra's Algorithm in C++, the robot will efficiently move materials between production lines, ensuring smooth and efficient manufacturing processes.

Objectives

The main objective is to replace the default path planner in Nav2 with a custom implementation of Dijkstra's Algorithm. This will enable the robot to autonomously navigate the factory floor, avoiding obstacles and optimizing its path for material transportation.

Tools and Technology

Results

The custom Dijkstra planner was successfully integrated into the Nav2 framework, and the robot demonstrated the ability to navigate the factory environment autonomously. Initial tests showed the planner correctly initialized and attempted to compute paths, although further refinement of the algorithm's implementation is required for optimal performance.

Key Insights
  • Algorithm Efficiency: The implementation of Dijkstra's Algorithm is more computationally intensive than the default planner, leading to longer path computation times.
  • Path Accuracy: The generated paths tend to be more grid-like, reflecting the nature of Dijkstra's Algorithm in a discretized environment.
  • Autonomous Navigation: Despite initial challenges, the project showcases the feasibility and benefits of custom path planning algorithms in industrial automation.

This project highlights the critical role of efficient path planning in robotics and demonstrates the practical application of theoretical algorithms in real-world scenarios.
